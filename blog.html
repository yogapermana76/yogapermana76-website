Pengalaman Masuk Hacktiv8

Hallo guys! Nama saya Yoga Permana, saya dari Bogor. Disini saya akan sedikit bercerita pengalaman saya untuk masuk Hacktiv8.

Oh iya, sebelumnya saya sekolah di SMK Taruna Terpadu 1 Bogor biasa disebut BORCESS - Bogor Center School (yang kata nya sekolah sejuta umat, siswa terbanyak di Jawa Barat bahkan di Indonesia ğŸ˜… hehehehe), saya dulu jurusan multimedia, lulusan tahun 2016.

Oke, langsung saja ceritanya, hehe.

Semenjak sekolah memang saya sudah bercita-cita ingin menjadi seorang developer atau dengan kata lain ingin menjadi programmer. Rencana nya saya ingin menggapai cita-cita saya dengan kerja terlebih dahulu kemudian ingin sambil melanjutkan sekolah di universitas. karena, saya ingin menghasilkan uang perbulan tapi bisa kuliah juga. Alhasil sampai sekarang saya sudah bekerja selama 2 tahun, dan rencana akan melanjutkan kuliah tahun ini.

Sebelumnya juga saya pernah mengikuti kursus web master di tebet (biar ga buta-buta amat setelah masuk kuliah jurusan IT). saya sudah belajar basic-basic dari html, css, php & mysql dan javascript juga. Tapi setelah menyelesaikan kursus tersebut saya kehilangan arah, soal nya yang diajarkan di tempat kursus itu hanya disuruh buat ini dan hasil nya nanti seperti ini, secara tidak langsung kita tidak tahu nanti nya kalo misalkan disuruh membuat website yang bisa diakses oleh semua orang, karena ga diajarkan logic di tempat kursus itu, tapi hanya mengikuti aja.

Setelah saya searchingÂ² "tempat kursus website terbaik di Jakarta dapat jaminan kerja", akhir nya saya menemukan Website nya Hacktiv8 di paling atas. Pas saya baca-baca ternyata hacktiv8 itu bukan tempat kursus ataupun universitas, tetapi coding bootcamp. Sebelumnya juga saya baru denger tentang coding bootcamp itu, akhirnya saya searching juga di mbah google ğŸ˜… (karena mbah google selalu tau hehehe). Setelah saya baca-baca, ternya benar coding bootcamp itu program nya bagus banget, dalam waktu kurang lebih 12 minggu kita bisa menjadi seorang developer profesional (dalam hati, sungguh luar biasa). Kemudian saya tertarik ingin gabung ke Hacktiv8 dan mencari tahu bagaimana cara gabungnya, dan ternyata untuk bisa gabung di Coding Bootcamp Hacktiv8 itu harus punya biaya besar. Dan disitulah saya mulai minder, saya berfikir: "mana mungkin saya masuk kesitu, karena biayanya besar banget, uang dari mana untuk biaya segitu? kalo sistem pembayarannya perbulan seperti universitas-universitas sih sanggup". Tapi, saya sangat tertarik banget ingin belajar di Hacktiv8 karena dalam waktu singkat kita bisa menjadi developer profesional. Tidak seperti kuliah, memakan waktu 4 tahun tapi belum tentu menjadi seorang developer yang sesuai standar perusahaan besar.

Pada suatu hari, ada informasi tentang beasiswa untuk belajar disitu, namanya kalo ga salah Hacktiv8-Schoolarship untuk menjadi insturktur di Hacktiv8, kesempatan besar bagi saya. Saya mencoba untuk daftar beasiswanya itu, dengan harapan agar bisa belajar di Hacktiv8. Setelah mendaftar, tidak lama kemudian ada email masuk dari admin Hacktiv8 mengenai jadwal testnya. Pas hari H nya saya langsung ke kampus Hacktiv8 di Pondok Indah, pas saya masuk ke ruangan testnya ternyata banyak banget yang ikut, bahkan ada kloter nya. Waktu itu saya kebagian kloter yang siang. Waktu itu testnya tentang logic dan soal terakhir translate bahasa inggris ke bahasa indonesia. Saya juga jadi minder, bagaimana mungkin saya bisa melewatinya dengan background saya hanya SMK, ditambah dengan jumlah peserta yang banyak. Tapi saya percaya diri aja, namanya juga usaha.

Beberapa minggu kemudian, ada pengumuman hasil tes tersebut. Dan hasilnya adalah tidak lolos ğŸ˜…. Tapi wajar sih, yaaa.. karena background saya hanya SMK dan kemampuan coding saya belum expert. Tapi itu semua menjadi pengalaman pertama saya mengikuti program beasiswa coding bootcamp di Hacktiv8. Oke, ga apa-apa kali ini gagal. Tapi kegagalan tersebut saya jadi kan pelajaran bagi saya pribadi, karena ingin menggapai cita-cita itu tidak mudah. (Tidak semudah itu Fergusooooo.... ğŸ˜… wkwkwkws).

Hari demi hari telah berlalu. Tiap minggu selalu dapat email dari Hacktiv8, saya baca-baca, ternyata menarik juga. Saya bertanya-tanya ke admin, mungkin adminnya sendiri bosan menjawab pertanyaanÂ² dari saya, wkwk, karena setiap hari selalu bertanya, bagaimana cara daftarnya, untuk biayanya apakah bisa dicicil, dll. Dan ternyata, untuk biayanya bisa dicicil. Wah, merasa lega banget ketika mendengar bahwa biayanya bisa dicicil ğŸ˜‚. Kalau bisa dicicil, mungkin saya bisa untuk mengikuti coding bootcamp itu. Nah, ketika itu saya mendapatkan info tentang event open house Hacktiv8, saya coba register untuk mendapatkan tiket masuknya. Beberapa minggu kemudian, tibalah acara open house itu. Beberapa menit kemudian acara open housenya pun dimulai. Saya menyimak sambutan dari instruktur dan pengalaman alumni Hacktiv8 selama belajar disitu. Ternyata Hacktiv8 itu keren banget, coding bootcamp pertama di indonesia, dan mempunyai kurikulum yang keren juga, dan jaminan kerja setelah lulus. Setelah pembukaan, saya dan yang lainnya, mengikuti study tour kampus Hacktiv8, mengelilingi kampus dan melihat murid-murid yang sedang belajar di lab. Pokoknya keren banget deh. Selesai acara study tour, ada sesi photo. Setelah itu, saya langsung pulang. Kemudian saya bercerita mengenai program tersebut kepada orang tua saya, dan memutuskan untuk gabung pada bulan November 2018, akhirnya saya resign dari tempat kerja saya demi gabung dengan Hacktiv8, wkwkwkwk.. pengorbanan banget gue ğŸ˜‚.

Pada pertengahan Oktober, saya memutuskan untuk daftar. Setalah mendaftar, minggu depannya saya di invite untuk hadir interview dan test. Sesi pertama yaitu interview, setelah selesai interview, bagian sesi mengerjakan soal, diberi waktu 40 menit. Soal tesnya hanya matematika(logic), bahasa inggris, dan essential. Untuk bahasa inggris, jawab soal dan translate ke bahasa indonesia. Dan Alhamdulillah testnya lancar tidak ada kesulitan, 90% yakin dengan jawabannya. Tinggal menunggu hasilnya. Setelah tes saya langsung pulang kerumah di Bogor. Pada hari seninnya, ada email masuk dari Hacktiv8. Saya buka dan baca isi emailnya, dan hasilnya Alhamdulillah diterima sebagai murid Hacktiv8.

Saya masuk batch 28 / Bold Fox. Dan kelas dimulai tanggal 10 November 2018 untuk phase 0, tepatnya hari senin. Sedangkan saya masih dikampung halaman, dan memutuskan untuk berangkat ke manggarai ke kontarkan bapak saya pada minggu malam..

Pesan saya : Jika kalian gagal masuk kampus impian atau apapun, jangan pernah berhenti untuk memperjuangkannya. Jangan menyerah begitu saja. Tetap Ikhtiar, sabar, berdoa dan belajar. Insha Allah, Allah akan memberikan kemudahan bagi kita. Karena Allah sudah menyiapkan yang terbaik buat kita semua.

Terimakasih telah membaca curhatan saya ğŸ˜…, semoga memotivasi, dan bermanfaat. Hehe. Mohon maaf jika ada kata yang kurang berkenan dihati kalian semua. Ini cerita real sesuai yang saya alami, bukan rekaan. Hehe.
